### Install TaLib
# http://mrjbq7.github.io/ta-lib/install.html
Ir a configuration/talib
$ tar xvf ta-lib-0.4.0-src.tar.gz
$ cd ta-lib
$ ./configure --prefix=/usr
$ make
$ sudo make install
